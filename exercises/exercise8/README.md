# Exercise8 開発環境構築（golang)

## はじめに

今後のエクササイズで必須となるGo言語環境をインストールします。
インストールの仕方はいろいろあるのですが、参考情報へのリンクの通りに導入してください。
その他の方法で導入してトラブルにあった場合は、その環境を削除していただき、再度インストールしていただきます。

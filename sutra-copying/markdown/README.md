# Markdownの写経

Markdownの写経元はGithubが提供しているドキュメントのサンプルが一番良いと思います。


タグを切り替えて、以下それぞれ写経してください。

- Text
- Lists
- Images
- Headers & Quotes
- Code
- Extras

https://guides.github.com/features/mastering-markdown/#examples

次のオンラインのMarkdwonエディタを使うと、入力と同時にMarkdownでの出力結果を見ることができます。
